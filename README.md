This project was developed during Ignite, course od the React Rocketseat, with the following technologies:

## :computer: How to run

  ```bash
    # clone repository
    git clone

    # enter web folder
    cd project-name

    # download dependencies
    yarn

    # running web project
    yarn dev
  ```
